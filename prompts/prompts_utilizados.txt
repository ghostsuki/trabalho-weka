PROMPTS UTILIZADOS PARA GERAÇÃO DO DATASET

Ferramenta utilizada:
ChatGPT (OpenAI)

Objetivo:
Gerar um dataset sintético para classificação de risco de evasão acadêmica em estudantes universitários.

------------------------------------------------------------
PROMPT 1 — Definição inicial do dataset
------------------------------------------------------------

"Gere um dataset sintético com 500 estudantes universitários para prever risco de evasão acadêmica.

Atributos:
- idade (17 a 45)
- frequencia (0 a 100)
- nota_media (0 a 10)
- horas_estudo (0 a 40)
- renda_familiar (800 a 15000)
- acesso_internet (sim/nao)
- faltas (0 a 60)
- extracurricular (sim/nao)

Classe alvo:
risco_evasao = baixo, medio ou alto"

Objetivo:
Criar a estrutura inicial do dataset.

------------------------------------------------------------
PROMPT 2 — Tornar o dataset mais realista
------------------------------------------------------------

"Torne o dataset mais realista:
- adicione desbalanceamento entre as classes;
- inclua padrões coerentes entre atributos e classe;
- mantenha consistência semântica."

Objetivo:
Simular comportamento próximo de dados reais.

------------------------------------------------------------
PROMPT 3 — Inserção de problemas reais
------------------------------------------------------------

"Adicione:
- 5% de valores faltantes;
- 5% de ruído;
- 3% de outliers."

Objetivo:
Simular problemas comuns em datasets reais.

------------------------------------------------------------
PROMPT 4 — Inserção de atributo irrelevante
------------------------------------------------------------

"Adicione um atributo irrelevante que não tenha relação com evasão acadêmica."

Resultado:
atributo `cor_favorita`

Objetivo:
Permitir etapa de seleção e remoção de atributos.

------------------------------------------------------------
PROMPT 5 — Formatação final
------------------------------------------------------------

"Exporte o dataset final em formato CSV compatível com Weka."

Objetivo:
Permitir importação direta no Weka.

------------------------------------------------------------
JUSTIFICATIVA
------------------------------------------------------------

A geração foi realizada de forma iterativa utilizando engenharia de prompts.

Cada prompt teve objetivo específico:
1. estruturar o problema;
2. tornar os dados realistas;
3. inserir ruído e inconsistências;
4. permitir pré-processamento;
5. garantir compatibilidade com a ferramenta Weka.

Esse processo assegurou que o dataset final atendesse aos requisitos do trabalho.
